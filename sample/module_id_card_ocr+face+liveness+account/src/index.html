데모 활용시 주의사항

• 신분증 이미지 사용 시 테스트데이터를 다운로드하여 테스트 하시기 바랍니다
• 수집된 데이터는 처리목적이 달성된 경우 지체없이 파기합니다
• 수집된 데이터는 테스트목적 이외의 용도로는 이용되지 않으며, 제3자에게 제공하지 않습니다
• 유스비는 직접적인 고유식별번호 수집 및 처리를 원칙적으로 금지합니다
